About Semantic Markup

You will hear the word semantic used often when you read or head about HTML. The core concept behind sematic markup is to ensure that you use the most meaningful HTML element available to describe the content you are marking up.

Text-Based HTML Elements

  • Headings
  • Paragraphs
  • Inline text element
  • Lists

For more information about HTML markup, visit the W3 Schools website.
